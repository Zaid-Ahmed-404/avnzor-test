# Avnzor Test

This is the repository for the **Avnzor Test** project, built with the CodeIgniter 4 framework.


## Getting Started

Follow these steps to set up the project locally:

### 1. Clone the Repository
```bash
git clone https://github.com/Zaid-Ahmed-404/avnzor-test.git
```

### 2. Install Composer
```bash
composer install
```

### 3. Environment Configuration
```bash
cp env .env
```
### 4. Create 'purchase' named database

### 5. Database Migrations and Seeders
```bash
php spark migrate
php spark db:seed ProductSeeder
```

### 6. Run the Project
```bash
php spark serve
```

The application will be accessible at: http://localhost:8080
